# KI-Kompass V1

## Enthalten
- Persönlicher KI-Kompass mit 6 Fähigkeiten
- KI-Reifegrad:
  - KI-Neuling
  - KI-Anwender
  - KI-Navigator
  - KI-Pilot
- KI-Energie statt täglichem Streak
- Session-Fortschritt `x/11`
- Fortschritt wird lokal im Browser gespeichert (`localStorage`)
- Kein Login, kein Backend, keine Datenbank nötig

## Integration
- Startseite: großer KI-Kompass-Block
- Session-Seiten: KI-Kompass + Session speichern Button
- Header/oben: dezenter Status-Chip
- `lektionen.html`: Kompass ebenfalls verfügbar
- `agent_dashboard.html`: Status-Chip verfügbar

## Hinweis
Der Fortschritt ist pro Browser/Gerät gespeichert. Für echte Accounts wäre später ein Backend nötig.
