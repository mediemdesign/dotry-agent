iPhone Fotos automatisch
sortieren mit Python

Tausende Fotos auf dem iPhone – und alles liegt durcheinander. Ein Python Script liest das Aufnahmedatum und den GPS-Ort direkt aus jedem Foto und sortiert alles automatisch in Ordner wie 2026-05 – Wien oder 2026-06 – Kreta. Einmal einrichten – nie wieder Foto-Chaos.

💡 Was ist EXIF? Jedes iPhone-Foto enthält unsichtbare Metadaten – Aufnahmezeitpunkt, GPS-Koordinaten, Kameraeinstellungen. Das Script liest diese Daten und nutzt sie zur automatischen Sortierung.

📋
Phase 2 · Nachmachen

Schritt für Schritt einrichten

1
Fotos vom iPhone auf den Computer kopieren
Mac: iPhone anschließen → Finder öffnen → iPhone auswählen → Fotos auf Desktop ziehen
Windows: iPhone anschließen → Explorer → iPhone → DCIM Ordner → kopieren

Alle Fotos in einen Ordner legen, z.B. Desktop/iPhone_Fotos
2
Python & Pakete installieren
Terminal öffnen und eingeben:
pip install Pillow requests
✅ "Successfully installed..." → weiter zu Schritt 3
⚠️ Mac: pip3 install Pillow requests
3
Script herunterladen
Das Script herunterladen und auf dem Desktop speichern:
📥 foto_sortierer.py herunterladen
4
Pfade anpassen (2 Zeilen im Script)
Das Script in einem Texteditor öffnen und diese zwei Zeilen anpassen:
foto_sortierer.py – Einstellungen 
# Wo liegen deine iPhone Fotos?
QUELL_ORDNER = "~/Desktop/iPhone_Fotos"

# Wo sollen die sortierten Fotos hin?
ZIEL_ORDNER = "~/Desktop/Fotos_Sortiert"
⚠️ Tilde (~) bedeutet dein Home-Verzeichnis. Alternativ: "/Users/deinname/Desktop/iPhone_Fotos"
5
Script starten!
Terminal öffnen, in den Ordner mit dem Script wechseln und starten:
cd ~/Desktop && python3 foto_sortierer.py
✅ Du siehst wie jedes Foto verarbeitet wird:
[1/247] IMG_0042.jpg ... ✅ → 2026-05 – Wien/
[2/247] IMG_0043.jpg ... ✅ → 2026-05 – Wien/
...

⚠️ Wichtig: Das Script kopiert deine Fotos – es verschiebt oder löscht nichts. Deine Originalfotos bleiben unverändert im Quellordner. Du kannst es also bedenkenlos ausprobieren.

Den Script anpassen

Im Script gibt es mehrere Einstellungen die du anpassen kannst:

foto_sortierer.py – Alle Einstellungen 
# Quell- und Zielordner
QUELL_ORDNER = "~/Desktop/iPhone_Fotos"
ZIEL_ORDNER  = "~/Desktop/Fotos_Sortiert"

# Welche Dateitypen sollen sortiert werden?
FOTO_TYPEN = {".jpg", ".jpeg", ".png", ".heic", ".heif"}

# GPS-Ortsnamen via Internet abrufen?
# True  = Wien, Kreta, Ansfelden (braucht Internet)
# False = nur Datum, kein Ortsname
ORT_ABRUFEN = True

# Vorhandene Fotos überspringen?
# True = keine Duplikate (empfohlen)
DUPLIKATE_UEBERSPRINGEN = True
