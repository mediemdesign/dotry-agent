Warum dieser Agent so stark ist

Viele KI-Kurse zeigen "Agent schreibt Blogartikel." Das interessiert die meisten nur begrenzt. Der Foto-Agent löst ein echtes Alltagsproblem das fast jeder kennt:

🔍
Fotos endlich wiederfinden
🏖️
Urlaub automatisch archivieren
👨‍👩‍👧
Familienfotos sauber sortieren
🏢
Firmenfotos strukturieren

Das Ergebnis ist sofort sichtbar, täglich nützlich – und macht den Nutzen von KI-Agenten greifbar.

Der Agent überwacht deine Foto-Quelle

Wähle woher deine Fotos kommen – der Workflow funktioniert mit allen drei:

iCloud Drive → Freigegebene Fotos
# iCloud Fotos in n8n einbinden
Trigger: iCloud Drive – Neue Datei
Ordner: ~/Library/Mobile Documents/com~apple~CloudDocs/Fotos
Filter: .jpg, .jpeg, .heic, .png
Dropbox → /Camera Uploads
# Dropbox Trigger in n8n
Trigger: Dropbox – Watch Folder
Ordner: /Camera Uploads
Ereignis: Neue Datei hochgeladen
Google Drive → Fotos Ordner
# Google Drive Trigger in n8n
Trigger: Google Drive – New File
Ordner: Meine Ablage / iPhone Fotos
MIME Type: image/jpeg, image/heic

Was der Agent mit jedem Foto macht

Jedes neue Foto durchläuft automatisch vier Schritte:

📋
Schritt 1
EXIF-Daten auslesen
Jedes iPhone-Foto enthält unsichtbare Metadaten. Der Agent liest sie automatisch aus:
DateTimeOriginal: 2026-05-12 14:32:07
GPSLatitude: 48.2484
GPSLongitude: 13.8382
Make: Apple
Model: iPhone 15 Pro
🗺️
Schritt 2
Ort erkennen via Reverse Geocoding
GPS-Koordinaten werden in echte Ortsnamen umgewandelt – kostenlos über OpenStreetMap:
48.2484, 13.8382
→
Gmunden
48.2093, 16.3728
→
Wien
35.3387, 25.1442
→
Kreta
📁
Schritt 3
Ordnerstruktur automatisch erzeugen
Der Agent legt Ordner nach dem Schema Jahr / Datum_Ort automatisch an:
📁 Fotos/
　　├── 📁 2026/
　　│　　├── 📁 2026-05-12_Gmunden/
　　│　　│　　└── IMG_0042.jpg
　　│　　├── 📁 2026-05-25_Wien/
　　│　　│　　└── IMG_0187.jpg
　　│　　└── 📁 2026-06-01_Salzburg/
　　│　　　　└── IMG_0299.jpg
✨
Schritt 4
Vision KI beschreibt das Bild
⭐ Premium Feature
Claude Vision oder GPT-4o analysiert das Foto und generiert automatisch einen beschreibenden Dateinamen. Das ist der echte "Wow-Moment":
📸 IMG_0042.jpg
(Hund am See)
→
2026-05-12_Gmunden_
Hund_am_Traunsee.jpg
📸 IMG_0187.jpg
(Geburtstagstorte)
→
2026-05-25_Wien_
Geburtstagstorte_Familie.jpg

💡 Wie funktioniert das? Das Foto wird an die Vision API geschickt mit dem Prompt: "Beschreibe dieses Foto in 3-5 Wörtern auf Deutsch. Nur Substantive, keine Artikel, mit Unterstrichen." Das Ergebnis wird direkt als Dateiname verwendet.

🔵 Block B – Nachmachen · n8n Setup
Den Foto-Agent selbst aufsetzen

Jetzt wird's konkret. In 20 Minuten läuft dein eigener Foto-Agent. Du brauchst nur einen Browser und eine E-Mail-Adresse. Kein Server, kein Programmieren – n8n macht das Setup einfach.

1
n8n Account erstellen – kostenlos
Geh auf n8n.io → "Get started for free" → mit Google oder E-Mail registrieren.
Der kostenlose Plan reicht für diesen Workflow vollständig aus.
✅ Du siehst den n8n Canvas – eine leere Arbeitsfläche
2
Workflow importieren
Den fertigen Workflow als JSON herunterladen und in n8n importieren:
📦
dotry.ai Foto-Agent Workflow
Fertiger n8n Workflow · 11 Nodes · GPS + Vision KI
⬇️ JSON herunterladen
In n8n importieren: Menü oben links → Import → From file → heruntergeladene JSON-Datei auswählen
✅ Du siehst alle 11 Nodes des Foto-Agents auf dem Canvas
3
Google Drive verbinden
Klick auf den ersten Node "📱 Neues Foto erkannt" → rechts "Credential" → "Create new" → Google Account verbinden → Ordner auswählen wo deine iPhone Fotos landen.
⚠️ Du musst denselben Google Account verwenden in den du deine Fotos lädst
✅ Der Node zeigt grün – Verbindung steht
4
Claude API Key eintragen (für Vision KI)
Klick auf den Node "✨ Vision KI – Bild beschreiben" → Credential → "Create new Anthropic credential" → API Key von console.anthropic.com einfügen.
⚠️ Ohne API Key funktioniert der Workflow ohne Vision KI – Dateinamen werden dann nur nach Datum und Ort erstellt
5
Ersten Test starten
Oben rechts auf "Test workflow" klicken → Ein Testfoto in deinen Google Drive Ordner hochladen → Der Workflow startet automatisch.
Du siehst wie jeder Node grün wird und was er verarbeitet. Am Ende liegt das Foto sortiert und umbenannt in deinem Zielordner.
✅ 2026-05-12_Gmunden_Hund_am_Traunsee.jpg – dein erster sortierter Agent-Output! 🎉
6
Workflow aktivieren – ab jetzt läuft es automatisch
Oben rechts den Toggle von "Inactive" auf "Active" stellen. Ab jetzt überwacht n8n deinen Ordner automatisch – jedes neue Foto wird sofort verarbeitet.
✅ Dein Foto-Agent läuft. Nie wieder Foto-Chaos. 🎉
